# Flask Blog Website

A full stack blog website built using Flask and Bootstrap.
Users can create, edit, and delete blog posts.

## Tech Stack
- Flask (Python)
- HTML, CSS, Bootstrap
- SQLite Database

## Features
- CRUD operations
- Responsive UI
- Simple design

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/
